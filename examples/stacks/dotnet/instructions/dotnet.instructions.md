# .NET Development Instructions

## Build & Test

- Build: `dotnet build`
- Test: `dotnet test`
- Watch: `dotnet watch run`

## Conventions

- Follow the conventions defined in the `dotnet-conventions` and `blazor-conventions` skills when active.
- Use `xunit-testing` skill patterns for test projects.
